## 三、组件设计

### 3.1 菜单按钮
- 外形：圆角 10px，宽 340px，高 52px
- 普通态：`COLOR_CARD` 底色，白色文字
- 选中态：蓝色边框 2px + 外发光 `box shadow` + 蓝色文字
- 悬浮态（EasyX 不支持 hover，改用选中态加亮）

### 3.2 信息面板
- 外形：圆角 10px，从 BOARD_TOP 延伸到底部
- 内部分区：模式名区 → 地图信息 → 分数卡片 → 道具区 → 操作提示
- 分数卡片：`COLOR_BG` 内嵌圆角卡片，大号金色数字
- 分割线：`COLOR_BORDER` 1px

### 3.3 棋盘
- 地面：`COLOR_BOARD` 纯色
- 墙壁：`COLOR_WALL` + 内阴影效果（可选）
- 网格线：`COLOR_BORDER` 半透明 1px（可选关闭）
- 圆角格子：食物/道具贴图绘制在带圆角的底框上

### 3.4 游戏结束画面
- 半透明深色遮罩覆盖棋盘
- 居中圆角卡片：结果显示 + 重新开始/返回菜单按钮
- 标题使用对应结果颜色（绿=胜、红=负、金=平）

## 四、动画与过渡

### 4.1 菜单切换
- 页面间淡入淡出：`BeginBatchDraw` 前画面保持，新画面一次性绘制
- 菜单项选择：选中项颜色从灰色渐变到蓝色（分 6 帧，每帧 50ms）

### 4.2 分数变化
- 吃食物时分数弹跳：字号从 28 → 36 → 28，持续 300ms
- 颜色从金色闪到亮白再恢复

### 4.3 暂停覆盖
- 半透明遮罩：深色底 + 60% 透明度矩形
- 文字淡入：从透明到白色，200ms

### 4.4 事件预警
- 预警文字脉冲：每隔 0.5s 字号在 26→30→26 之间切换
- 轰炸区边界：预警时边框闪烁（橙色交替）

### 4.5 蛇死亡
- 蛇身红色闪烁 3 次（每 200ms 切换红色/原色）
- 棋盘整体闪红一次

## 五、粒子系统

### 5.1 粒子结构
```c
#define MAX_PARTICLES 200

typedef struct Particle {
    bool active;
    float x, y;          // 屏幕坐标
    float vx, vy;        // 速度
    int lifeMs;          // 剩余生命
    COLORREF color;
    int radius;          // 半径（逐渐缩小）
} Particle;
```

### 5.2 粒子更新
每帧调用 `updateParticles(deltaMs)`:
- 遍历活跃粒子，更新位置 `x += vx * dt`，`y += vy * dt`
- 递减 lifeMs，lifeMs < 500 时 radius 缩至 1
- lifeMs <= 0 时 deactivate
- 渲染：`solidcircle(x, y, radius)`

### 5.3 粒子触发事件
| 事件 | 触发函数 | 粒子数 | 颜色 | 生命 |
|------|----------|--------|------|------|
| 吃普通食物 | `spawnEatParticles` | 6 | 金色 (255,202,99) | 600ms |
| 吃得分类食物 | `spawnEatParticles` | 10 | 亮绿 (124,255,107) | 800ms |
| 蛇死亡 | `spawnDeathParticles` | 20 | 红色 (255,64,64) | 1000ms |
| 轰炸爆破 | `spawnBombParticles` | 30 | 橙红 (255,100,30) | 1200ms |
| 箭矢命中 | `spawnHitParticles` | 8 | 亮白 (255,255,200) | 400ms |

### 5.4 菜单背景粒子
- 始终活跃的少量飘浮粒子（最多 15 个）
- 颜色：半透明蓝 (88,166,255, 50% 亮度)
- 速度：缓慢上飘 `vy = -0.3`，左右随机漂移
- 生命：4000ms，到底部时重生到顶部

## 六、随机事件 UI 强化

### 6.1 轰炸区
- 区域边界：额外 2px 红色虚线或实线边框
- 预警时：区域边缘闪烁（交替红色/橙色，每 0.5s）
- 爆破时：区域内部绘制随机火花粒子

### 6.2 箭雨
- 发射源箭头更明显：黄色方块 + 红色箭头 + 白色描边
- 箭矢飞行拖尾：箭矢身后绘制 2-3 个渐淡的半透明圆点

### 6.3 侧栏倒计时
- 事件名下方显示进度条（`eventTimerMs / 12000 * 进度条宽度`）
- 最后 3 秒进度条变红

## 七、性能策略

| 地图 | 粒子上限 | 箭头拖尾 | 说明 |
|------|----------|----------|------|
| 20x20 | 200 | 开 | 全速 |
| 50x50 | 200 | 开 | 全速 |
| 100x100 | 100 | 关 | 降级，禁用拖尾 |

性能监控：每 10 秒检查平均帧时间。若 > 40ms，自动降级到等级 2（禁用粒子、保留动画）。

## 八、实现优先级

1. **Pass 1 — 静态美化**：配色/字体/圆角/卡片 → 即时生效
2. **Pass 2 — 动画过渡**：菜单切换/分数弹跳/预警脉冲/死亡闪烁
3. **Pass 3 — 粒子系统**：粒子结构 + 更新渲染 + 各事件触发
4. **Pass 4 — 事件 UI 强化**：轰炸边框/进度条/箭矢拖尾/性能降级
